# ubnutu 16.04 系统安装

本次给大家带来的是关于 ubnutu 系统的安装方法，首先到 ubnutu 的中文官网下载除非电脑配置真的不高否则请选择 64 位的版本下载

下载完成之后将会有以下这个 w 文件名为 ubuntu-16.04.2-desktop-amd64.iso 镜像文件!
本次是在虚拟机中安装的，所以还需要到下载一个vmware workstation 12的软件（由于商业软件的原因所以要自己baidu查找破解版）
接下来打开 vmware 　 workstation 如下图 选择文件点击新建虚拟机 点击下一步 继续点击下一步（这时候不建议使用浏览选择刚刚下载的 ubnutu 镜像文件，选择的话会自动安装的- -还是建议手动安装好）

在这一步选择所下载系统的选项。（我们下载的是 ubnutu64 位的）

继续下一步这时选择存放虚拟机硬盘的目录（就是一个文件来的- -）

接下这选择虚拟机的硬盘大小（自己觉得要多少就写多少，就算你写多点也没问题因为这可以看做是一个上限值）如图：我选择的是 50G。

剩下的默认就可以了，点击完成后 在 vmware 　 workstation 的左边你将会看到刚刚新建的虚拟机（本文新建的虚拟机为 ubnutu64）如图

接下来点击编辑虚拟机把刚刚的 iso 挂载到虚拟机的光驱里。

回到主界面点击开启虚拟机，开始安装系统（激动吧。）

选择中文按回车键

选择安装的选项（进入之后依然是选择中文- -）

勾选安装第三方软件（就是等于安装了 window 的驱动），然后继续下一步。

新手直接勾选多 USE LVM (为了以后学习的方便)，然后选择现在安装就可。

选择继续（这时候就是格式化你的磁盘）

选择时区并继续。

选择键盘的语言（中文不解释），然后继续

输入用户名密码之后选择继续。

看到这张图的时候证明已经在安装了（慢慢等吧）

最后重启下就行了。
